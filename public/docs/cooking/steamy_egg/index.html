    계란찜#

    1-2인 분#

    재료#

    - 달걀 2개 
- 양파 조금 
- 쪽파 조금 
- 모짜렐라 치즈 50g 
- 체다치즈 1장 
- 맛술 1큰술 
- 참치액 또는 굴소스 1큰술
- 물 35ml
- 우유 35ml

    조리 순서#

    1. 계란을 잘 풀어준다 
2. 치즈를 재외한 재료를 넣고 섞는다 
3. 전자렌지에 1분30초 데운다
4. 체다 치즈를 넣고 섞은후 모짜렐라 치즈를 위에 뿌려준다
3. 전자렌지에 1분30초 데운다
