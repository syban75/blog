    계란 볶음밥#

    3인 기준#

    재료#

    - 식은밥 2인분
- 달걀 6개
- 대파 많이
- 당근 적당히
- 간장 한스푼
- 소금 적당히
- 굴소스 조금
- 후추 조금
- 참기름 조금
- 식용유

    조리 순서#

    1. 계란을 잘 풀어준다
2. 대파를 얇게 썰어 기름에 볶아 파기름을 내고, 당근을 넣어 볶는다
3. 계란 푼 물을 넓게 펴서 넣는다
4. 간장은 팬 가장자리에 둘러 넣고, 소금, 굴소스를 넣는다.
5. 밥을 넣고 잘 섞으면서 볶는다
6. 후추와 참기름을 넣고 마무리로 볶는다
